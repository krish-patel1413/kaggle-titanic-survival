# Kaggle Titanic Survival Prediction

This repository contains my machine learning model for the classic **Titanic: Machine Learning from Disaster** competition on Kaggle. 

##  Performance
* **Public Leaderboard Score:** 0.7000 (70% Accuracy)
* **Model Used:** Random Forest Classifier

##  Overview
The goal of this project is to predict which passengers survived the Titanic shipwreck using machine learning tools in Python. 

### Features Used:
* `Pclass` (Passenger Class)
* `Sex` (Gender)
* `SibSp` (Number of Siblings/Spouses Aboard)
* `Parch` (Number of Parents/Children Aboard)

## Tech Stack & Libraries
* **Language:** Python 3
* **Environment:** Kaggle Notebooks
* **Libraries:** `pandas`, `numpy`, `scikit-learn`
